\doxysubsection{Detailed Description}
Klasa do generowania liczb losowych w określonym zakresie. 

Klasa zawiera metody umożliwiające generowanie liczb pseudolosowych w zadanym przedziale. Korzysta z mechanizmu inicjalizacji generatora za pomocą ziarna.
